Minimalist harmony sounds different not because of exotic scales but because of voicing choices. Dense triads "land" — they have a root-third-fifth logic that pulls toward function. Open voicings hover. This chapter is the harmonic vocabulary behind the sound: what notes, which order, which registers, and why C# is mostly off-limits in D Dorian.
The Problem with Dense Triads

A close-position D minor triad (D3–F3–A3) is three notes stacked in thirds with no gaps. It's acoustically complete — the ear hears it as a resolved, named thing. That sense of "named completion" is exactly what minimalism avoids.

Instead, minimalist harmony uses voicings that don't resolve to a clear chord identity: open 5ths (D + A, no third), sus2 (D + E + A), sus4 (D + G + A), added 9th (D + F + A + E). These voicings hover — the ear can't decide if they're major, minor, or something else entirely.
Why C# Is (Mostly) Off-Limits

In D harmonic minor, C# sits a half step below D — a powerful magnetic pull. The ear hears it and waits for D. That waiting is functional harmony: tension demanding resolution. Minimalism is about removing that demand.

D Dorian uses C natural instead of C#. C natural is a whole step below D, which doesn't create the same urgency. The ear can rest on a C without feeling compelled to resolve it upward.

The rule: avoid C# unless you explicitly want tonal gravity at that moment. A C# can be a powerful arrival tool precisely because you've avoided it everywhere else — use it sparingly.
D + A: The Ambiguous Core

The open 5th — D and A with no third — is the most neutral harmonic statement available. The ear cannot determine major or minor. Arvo Pärt builds entire pieces on open 5ths, using their ambiguity to create a timeless, bell-like quality.

In practice: D2 in the bass, D3 + A3 above. Or D1 + D2 + A2 for a thicker version. The third (F) is deliberately absent. Adding it would commit the ear to minor; the open 5th commits to nothing.

This isn't emptiness — it's space. The listener's imagination fills in the missing third differently on each hearing.
Register Logic for Minimalist Voicings

Minimalist voicings follow a consistent register logic: bass notes carry the root, middle register carries the structural 5th, and color notes float in the upper register. Color notes — extensions like the 7th, 9th, or 4th — only work when placed above the structural core.

A typical stack: D2 (bass drone) → D3 + A3 (open 5th in the middle) → F4 + C5 (color notes floating above). The color notes are heard as additions, not as the chord's identity.

The rule: never put color notes in the bass. A C5 floating above a D2 bass reads as an added 7th. A C2 in the bass with D above reads as a C chord with an added 9th — completely different harmonic meaning.
The Deliberate Break from Bach

Bach forbids parallel fifths and octaves — they collapse voice independence into a single thick line, erasing the counterpoint's reason for existing. In minimalism, that collapse is the point.

Parallel motion — both voices moving in the same direction by the same interval — creates a homogenous, wash-like texture. Philip Glass's keyboard music (Metamorphosis, Glassworks) is built on it. Two hands, same intervals, moving together.

Use parallel motion when you want that immersive, undifferentiated wash. Use contrary or oblique motion when you want independence and tension. Both choices are valid — the difference is intentional.

Cross-reference: For Bach's rules against parallel fifths/octaves and the reasoning behind them, see Ch 03 — Two Voices. Understanding why Bach forbids them makes the minimalist choice to use them deliberate rather than ignorant.
Add a Chord Track

Open your M09 session. You have five active layers: Hocket A, Hocket B, Phase B, the M08 poly pattern, and the pulse.

Create a new MIDI track — use a pad or sustained synth timbre (longer attack, long sustain, slow release). Name it "M10 chords".

Create a 4-bar chord clip:

  • Bar 1–2: D open 5th — D3 + A3, half notes (two notes per bar)
  • Bar 3–4: Gsus4 — G3 + C4 + D4, half notes

This two-chord vamp (Dm i → Gsus4 IV) is the D Dorian fingerprint. The G chord's B natural (implied by the Gsus4 context) against the D chord's C natural creates the characteristic modal color.

Balance the mix: Play all existing tracks against the new chord layer. The chords should sit underneath the hocket melody — reduce chord velocity to around 60–70 if it competes. The hocket voices should remain primary.

You now have six layers: pulse, Phase B, poly pattern, Hocket A, Hocket B, and a sustained chord vamp. The chord layer gives the floating hocket notes a harmonic home without locking them into functional progressions.

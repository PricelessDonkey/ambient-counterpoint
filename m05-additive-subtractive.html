CHAPTER 05 ◇ ADDITIVE & SUBTRACTIVE PROCESS ◇ CHAPTER 05 ◇ ADDITIVE & SUBTRACTIVE PROCESS ◇ CHAPTER 05 ◇ ADDITIVE & SUBTRACTIVE PROCESS Additive & Subtractive Process Building and dissolving: the loop as a process, not a product. Reich's additive process turns composition into a game with one rule: add one note per cycle. This chapter shows how to grow a texture from silence to density — and then take it back apart — using nothing but clip duplication and careful note ordering. The process answers "what next?" so you don't have to.

Start with one note. Repeat it N times. Add one note. Repeat. Add one more. Continue until the texture is full. The additive process turns the act of composition into a rule: the music grows from almost nothing to complete density, and the listener hears every step of the journey.

This is the technique behind Reich's Drumming and many of Glass's keyboard works — the process itself is the composition. You don't need to decide what happens next: the rule decides for you.

Why it works

Repetition builds expectation. When a new note arrives, the ear hears it against the accumulated pattern and experiences it as arrival — even a single added pitch feels like an event. The longer you wait before adding, the more weight each addition carries.

The reverse journey: start with the full texture and remove one note every N repetitions. Subtractive process tends to feel more emotionally powerful than addition — something is being taken away. The listener experiences loss, not gain.

Silence earned by subtraction has more weight than silence from the start. By the time you arrive at a single note, that note carries the memory of everything that preceded it.

What to remove first

Remove color notes first (the ones farthest from the root/fifth), and save the root for last. This mirrors tonal gravity: the center survives while the ornamentation dissolves. Reversing this (removing root first) creates disorientation — use intentionally.

The order of addition and removal is a compositional decision. The rule tells you how many notes to add each cycle; you still choose which note and when the cycle turns.

Addition order — ascending scale

Add notes in ascending scale order from the root: root (D), then third (F), then G, then fifth (A). This creates a smooth, stepwise contour — each addition is the next note up the scale, so the texture grows naturally rather than leaping around. The listener hears it as a gradual filling-in.

Cycle length — how patient are you?

A cycle of 4 repetitions per clip feels fast and mechanical. 8 or 16 repetitions feels meditative. 32+ begins to feel like actual time passing. Most Reich and Glass additive passages use 8–16 repetitions per stage. Match the cycle length to the tempo: at 100 BPM, 8 bars of 4/4 = about 20 seconds per stage.

Subtraction order — color notes first

When subtracting, remove color notes first — the notes that create harmonic interest rather than the tonal backbone. From D F G A, subtract in this order: G → F → A → D. Color note G goes first; the fifth (A) survives until nearly the end; the root (D) is last. The final D has earned its isolation.

Two ways to build additive/subtractive sequences in Ableton Live:

Method 1: Separate clips (recommended)

Create one MIDI clip per stage. Each clip contains one more (or one fewer) note than the previous. Arrange these clips sequentially in Arrangement View, each repeated N times. This is the cleanest method: every stage is explicit, easy to edit, and can be different lengths.

Method 2: Note mute automation

Use a single clip with all notes present. In the MIDI clip editor, draw velocity automation or use the note-mute feature to silence notes at specific bars. Less transparent than Method 1 but useful if you want the option to vary the rhythm of existing notes while adding new ones.

Clip naming convention

Name clips clearly: M05-A1 [D], M05-A2 [D F], M05-A3 [D F G], M05-A4 [D F G A]. The bracket notation makes the clip contents readable at a glance in the Arrangement timeline without having to open each clip.

You don't need to be mechanically strict. Semi-process uses the additive/subtractive structure as a skeleton and makes expressive choices within it. This is Glass's approach: the process provides the answer to "what next?" but the composer still controls dynamics, articulation, and pacing.

What the rule gives you

The rule eliminates decision fatigue. Because you know what will change next, you can focus entirely on when to change it and how to prepare the listener. The constraint is liberating: it narrows the infinite possibility space to a single dimension.

Expressive choices within the rule

Vary the number of repetitions per stage (don't always use 8). Introduce a dynamic swell before adding a new note — make the arrival feel earned. Allow a stage to last longer than planned if it feels right. The rule says add one note; it doesn't say when.

Cross-reference: additive process is the minimalist analogue of Fortspinnung — spinning material out from a seed. For the Bach counterpart and how a sentence grows from its opening gesture, see Ch 07 — The Sentence.

The process answers "what next?" — your job is to answer "when." Strict or semi-strict, the rule gives you forward motion without requiring constant invention.

Open your M04 session (100 BPM, melody clips A/A′/A″, pulse track). You'll build a 32-bar additive-then-subtractive sequence on a new track.

Step 1 — Create four additive clips

Add a new MIDI track. Create four 1-bar clips:

  • M05-A1 [D]: D4, quarter note, beat 1 only.
  • M05-A2 [D F]: D4 beat 1, F4 beat 3.
  • M05-A3 [D F G]: D4 beat 1, F4 beat 3, G4 beat 2.
  • M05-A4 [D F G A]: D4 beat 1, F4 beat 3, G4 beat 2, A4 beat 4.
Step 2 — Arrange in Arrangement View

Place each clip repeated 4 times (4 bars each): A1 × 4 bars → A2 × 4 bars → A3 × 4 bars → A4 × 4 bars. That's 16 bars of addition. Then reverse the sequence for the subtractive return: A3 × 4 → A2 × 4 → A1 × 4 = 12 more bars. Total: 28 bars. Add 4 bars of silence at the end for arrival weight.

Step 3 — Listen with the pulse

Keep the M04 pulse track running throughout. Notice how the pulse gives each addition a rhythmic context — D alone sounds sparse against a pulse; D+F+G+A sounds full. The pulse is the constant that makes the addition audible.

Carry-forward: you now have 4 additive clips (M05-A1 through M05-A4) arranged in a 32-bar arc. Keep them — next chapter's phase work will run a duplicate of this track against itself.
